# La API de validacion de restricciones #

(The constraint validation API)

Disponible en los elementos:

-HTMLButtonElement
-HTMLFieldElement
-HTMLInputElement
-HTMLOutputElement
-HTMLSelectElement
-HTMLTextAreaElement

Propiedades

-validationMessage: 
-validity
-willValidate

Metodos

-checkValidity()
-setCustomValidity(message)
-HTMLFormElement.reportValidity
